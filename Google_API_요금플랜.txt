===============================================
           현재 사용 중인 Google API
===============================================

🎙️ Google Cloud Speech-to-Text API
💰 요금 구조:
   - 기본 요금: 분당 $0.006 (약 6원)
   - 무료 할당량: 월 60분 무료 제공
   - 추가 사용량: 무료 할당량 초과 시 분당 $0.006

📈 사용량 계산:
   - 현재 프로그램: 15초 녹음 × 인식 횟수
   - 예시: 하루 10회 녹음 = 150초 = 2.5분
   - 월 30일 기준: 75분 (무료 할당량 60분 초과)

📊 Google Sheets API
💰 요금 구조:
   - 완전 무료: 사용량 제한 없음
   - 일일 할당량: 1,000,000 요청/일
   - 현재 사용량: 매우 적음 (셀 업데이트만)

💾 Google Drive API
💰 요금 구조:
   - 완전 무료: 기본 사용량 무료
   - 일일 할당량: 1,000,000 요청/일
   - 현재 사용량: 스프레드시트 접근만 (매우 적음)

===============================================
              비용 절약 팁
===============================================

1. Speech-to-Text 최적화:
   - 녹음 시간 단축 (15초 → 10초)
   - 불필요한 녹음 방지
   - 월 60분 무료 할당량 내에서 사용

2. 현재 예상 비용:
   - 월 사용량: 약 75분
   - 추가 비용: (75-60) × $0.006 = $0.09 (약 120원)
   - 연간 비용: 약 $1.08 (약 1,440원)

===============================================
              모니터링 방법
===============================================

Google Cloud Console에서 사용량을 실시간으로 모니터링:
- Billing → Usage → Speech-to-Text API
- 월별 사용량 및 비용 확인 가능
